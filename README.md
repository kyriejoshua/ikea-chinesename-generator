
## Installation

```shell
npm install ikea-chinesename-generator --save
yarn add ikea-chinesename-generator
```

## Usage

* Javascript
```javascript
import generateIkeaLikeName from 'ikea-chinesename-generator';

const ikeaName = generateIkeaLikeName();
console.info(ikeaName);
```
